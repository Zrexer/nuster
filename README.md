# nuster

UnderDomain Finder

# Usage

Read on file:
```
nuster -u https://google.com(/) -o #<fileName>
```

Simple list:
```
nuster -u https://google.com(/)
```

Get help:
```
nuster
```
or 
```
nuster -h
```
or 
```
nuster --help
```
